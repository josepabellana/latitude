## Introduction
A mixed chart combines different chart types (for example, bars and lines) to present complementary data and give more context in a single visualization.

## Syntax

You can use either `MixedChart` or `MixedChartData` to visualize query results in a chart.

Use **MixedChart** when you want the component to automatically handle the query fetching.

Use **MixedChartData** when you already fetched the data using [useQuery](/react/running-query/use-query).

<CodeGroup>
```tsx MixedChart
import { MixedChart } from '@latitude-data/react';

function MyComponent() {
  return (
    <MixedChart
      queryPath='titles'
      params={{
        start_year: 2010,
        end_year: 2014
      }}
      x='release_year'
      y={[
        {name: 'count_shows', chartType: 'bar'},
        {name: 'count_movies', chartType: 'line'}
      ]}
      xTitle='Year'
      yTitle='Titles'
    />
  );
}
```
```tsx MixedChart
import { MixedChartData, useQuery } from '@latitude-data/react';

function MyComponent() {
  const { data, isLoading } = useQuery({
    queryPath: 'titles'
    params: {
      start_year: 2010,
      end_year: 2014
    }
  });

  return (
    <MixedChartData
      data={data}
      isLoading={isLoading}
      x='release_year'
      y={[
        {name: 'count_shows', chartType: 'bar'},
        {name: 'count_movies', chartType: 'line'}
      ]}
      xTitle='Year'
      yTitle='Titles'
    />
  );
}
```
