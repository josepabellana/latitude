## Introduction
The Bar chart represents quantitative data with vertical or horizontal bars, showing comparisons across categories or time. It highlights differences in values, making it ideal for comparing multiple datasets and identifying trends or outliers.

## Syntax

You can use either `BarChart` or `BarChartData` to visualize query results in a chart.

Use **BarChart** when you want the component to automatically handle the query fetching.

Use **BarChartData** when you already fetched the data using [useQuery](/react/running-query/use-query).

<CodeGroup>
```tsx BarChart
import { BarChart } from '@latitude-data/react';

function MyComponent() {
  return (
    <BarChart
      queryPath='titles'
      params={{
        start_year: 2010,
        end_year: 2014
      }}
      x='release_year'
      y={[
        { name: 'count_shows' },
        { name: 'count_movies' }
      ]}
      xTitle='Year'
      yTitle='Titles'
    />
  );
}
```
```tsx BarChart
import { BarChartData, useQuery } from '@latitude-data/react';

function MyComponent() {
  const { data, isLoading } = useQuery({
    queryPath: 'titles'
    params: {
      start_year: 2010,
      end_year: 2014
    }
  });

  return (
    <BarChartData
      data={data}
      isLoading={isLoading}
      x='release_year'
      y={[
        { name: 'count_shows' },
        { name: 'count_movies' }
      ]}
      xTitle='Year'
      yTitle='Titles'
    />
  );
}
```
