## Introduction
The Scatter chart displays data points on a grid, representing variable values with dots. It's excellent for revealing correlations, distributions, and outliers across datasets, making it crucial for statistical analysis and trend identification.

## Syntax

You can use either `ScatterChart` or `ScatterChartData` to visualize query results in a chart.

Use **ScatterChart** when you want the component to automatically handle the query fetching.

Use **ScatterChartData** when you already fetched the data using [useQuery](/react/running-query/use-query).

<CodeGroup>
```tsx ScatterChart
import { ScatterChart } from '@latitude-data/react';

function MyComponent() {
  return (
    <ScatterChart
      queryPath='titles'
      params={{
        start_year: 2010,
        end_year: 2014
      }}
      x='imbd_score'
      y='runtime'
    />
  );
}
```
```tsx ScatterChart
import { ScatterChartData, useQuery } from '@latitude-data/react';

function MyComponent() {
  const { data, isLoading } = useQuery({
    queryPath: 'titles'
    params: {
      start_year: 2010,
      end_year: 2014
    }
  });

  return (
    <ScatterChartData
      data={data}
      isLoading={isLoading}
      x='imbd_score'
      y='runtime'
    />
  );
}
```
